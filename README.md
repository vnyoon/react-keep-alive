## Context
  * 上下文 -> Provider -> 子组件们提供公共的属性或方法；
  * 缓存的是什么？
    - 真实DOM；
  * 组件的视图渲染？
    - 一定建立在append 真实DOM的基础上；
  * 缓存是怎么做的？
    - 保存在一个对象：id：{ nodes, ReactElement, status }

## Provider
  * 通过方法、属性，组件拿到相关的nodes设置缓存的方法和渲染；
